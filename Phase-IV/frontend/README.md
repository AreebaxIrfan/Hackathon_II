# AI Todo Agent Frontend

Frontend for the AI-powered todo assistant using OpenAI ChatKit.

## Tech Stack

- Next.js 16+
- React 19+
- OpenAI Chat Components
- TypeScript

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

## Components

- Chat interface using ChatKit
- Connection to backend API
- User authentication integration
